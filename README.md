# Kopai Integration Examples

Working examples for integrating with [Kopai](https://kopai.app) observability platform using OpenTelemetry.

## Examples

| Example | Description | Guide |
|---------|-------------|-------|
| [node-js](./node-js) | Backend Node.js with auto-instrumentation | [Node.js Integration](https://docs.kopai.app/integration/node) |
| [java](./java) | Backend Java with OpenTelemetry agent | [Java Integration](https://docs.kopai.app/integration/java) |
| [react-spa](./react-spa) | React frontend + Express backend | [Browser Integration](https://docs.kopai.app/integration/browser) |

## Quick Start

1. Get your API token from [kopai.app/settings](https://kopai.app/settings)
2. Clone this repo
3. Follow the README in each example directory
